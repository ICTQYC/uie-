# UIE模型微调项目

本项目基于 UIE（Universal Information Extraction）模型，针对特定任务进行了微调。适合用于信息抽取场景，项目内容包括模型微调脚本、数据集样例和相关配置。

## 项目简介

- **模型**: UIE（通用信息抽取模型）
- **任务类型**: 信息抽取（可定制用于抽取实体、关系等）
- **开发背景**: 研一实验室组内项目，主要目标是掌握模型微调流程并实现应用

## 环境依赖

- Python >= 3.8
- torch >= 1.10.0
- transformers >= 4.20.0
- 其他依赖见 `requirements.txt`
